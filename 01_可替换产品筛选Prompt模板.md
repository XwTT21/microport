# 可替换产品筛选 Prompt 模板（无LibreOffice）

```text
你现在要在本地文献工作区中，为【产品/器械名称】做文献筛选、适应性评估、数据摘录、Excel工作簿和Word报告。

注意：当前目标电脑没有LibreOffice，不能完成DOCX页面PNG视觉QA。你只能做DOCX结构检查，并在最终回复中明确说明这个限制。

工作路径：
【填写本地路径】

基础资料：
- 优先复用：已有的基础文献清单、题录表或上一轮整理结果
- 如果缓存不存在，再从原始文件夹读取：
  - Cochrane
  - Google Scholar
  - PubMed
  - 文献2次搜索(24-26.6)
- 默认不纳入：产品文献

本次主题：
【产品/器械名称】

强关键词：
- 【英文强关键词1】
- 【英文强关键词2】
- 【缩写】
- 【中文同义词】

弱关键词：
- 【只在上下文成立时使用的词】
- 【产品类别泛称】

纳入标准：
1. 题名或摘要明确显示【产品/器械】是研究对象、干预、对照、核心器械、技术平台或主要临床问题。
2. 允许研究类型：
   - 【临床原始研究是否纳入】
   - 【系统综述/Meta/指南是否纳入】
   - 【体外/工程/性能测试是否纳入】
   - 【经济学/资源/环境研究是否纳入】
   - 【病例报告是否纳入】
3. 至少记录一个与【产品/器械】相关的安全、性能、有效性或资源结局。

排除标准：
1. 其他器械/术式为核心主题，且【产品/器械】只是背景。
2. 【产品/器械】仅在方法中顺带出现。
3. 非目标疾病/用途。
4. PDF不可读或题录不足时，放入“待人工确认”。

必须输出：
- Excel工作簿
- Word报告
- 筛选统计
- 分层数量
- 复用资料说明
- 明确说明：无LibreOffice，未完成DOCX页面视觉QA
```
