## 判断规则

一条文献纳入必须满足：

1. 题名或摘要显示本产品是核心主题。
2. 研究问题与目标用途一致。
3. 有可摘录结局。
4. 资料可审计。

放入待确认：

- 只有全文提到产品。
- 题名很宽泛但摘要可能相关。
- PDF不可读。
- 题录匹配低置信。
- 结局不明确。

排除：

- 产品只是背景或方法工具。
- 研究对象是其他器械/术式。
- 非目标疾病/用途。
- 产品文献目录默认排除。

## 分层模板

| 分层 | 含义 |
|---|---|
| Similar device clinical dataset | 直接临床证据 |
| SOTA dataset | 综述、Meta、指南、技术现状 |
| Technical supportive data | 体外、工程、性能测试 |
| Economic/resource evidence | 成本、维修、灭菌、资源、环境 |
| Supportive case evidence | 病例或特殊场景 |
| Pending confirmation | 待人工复核 |

## D/A/P/R/T/O/S/C模板

每个产品都要重新定义，不要照搬旧产品。

| 维度 | 判断问题 |
|---|---|
| D | 是否直接评价目标产品或同类器械 |
| A | 用途是否一致 |
| P | 人群是否适用 |
| R | 报告是否完整可审计 |
| T | 证据类型是否可用 |
| O | 是否有相关结局 |
| S | 是否有统计支持 |
| C | 是否有临床意义 |
